# Next.js App

Minimal Next.js (TypeScript) app scaffolded for the DevSecOps portfolio.

Run locally:

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
npm start
```

CI trigger: pushed changes on development to run Next.js CI (security scans + build).
